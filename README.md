# Cleaning-and-merging-in-R
This is my very first real practice of data cleaning using R. I used functions like str(), join(), mutate(), etc. 
## Project overview
This project demonstrates data cleaning and integration of three datasets using R.

## Objectives
- Clean missing values
- Standardize variable names
- Merge three datasets
- Produce an analysis-ready dataset
- Export the clean dataset table as a word file

## Tools
- R
- dplyr
- tidyr
- flextable

## Skills demonstrated
- Data cleaning
- Data transformation
- Data merging (left_join, inner_join, etc.)
- Handling missing values
